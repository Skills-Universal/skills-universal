## Contenuto

- **6-dimension Visual Hierarchy Analysis:** Contrast, Scale, Spacing, Color, Typography, Layout Flow
- Color theory: hue, saturation, value, color relationships, semantic meaning
- Typography: typeface selection, scale, line height, letter spacing, pairing
- Visual hierarchy: size, weight, color, position, whitespace
- Layout systems: grid, alignment, balance, proportion
- Component visual standards: buttons, cards, forms, navigation
- Design system tokens: spacing scale, color palette, type scale
- Anti-AI-aesthetic checklist
- Review framework: how to critique a design systematically

## Quando caricare

Colori non funzionano, tipografia fuori luogo, gerarchia non chiara, layout congestionato, qualsiasi critica visiva.
